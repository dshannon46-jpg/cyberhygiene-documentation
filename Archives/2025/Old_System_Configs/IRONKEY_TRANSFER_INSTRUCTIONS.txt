## Step 1: Unlock IronKey Vault Privacy 50

### On Windows:
1. Connect IronKey USB drive to Windows computer
2. Wait for virtual CD-ROM to mount
3. Open the IronKey drive and run "IronKey.exe"
4. Enter your IronKey PIN/password
5. The encrypted partition will unlock and appear as a new drive

### On macOS:
1. Connect IronKey USB drive to Mac
2. Wait for virtual CD-ROM to mount
3. Open the IronKey volume and run "IronKey.app"
4. Enter your IronKey PIN/password
5. The encrypted partition will unlock and mount

### On Linux (if supported):
1. May require unlocking on Windows/Mac first
2. Then reconnect to Linux - partition should remain unlocked

## Step 2: Transfer Recovery Image to IronKey

### Option A: Copy Archive Directly
```bash
# Verify IronKey encrypted partition is mounted
df -h | grep -i iron

# Copy the archive
cp CyberHygiene_Recovery_Image_2025-12-18.tar.gz /media/IronKey/

# Verify checksum
cd /media/IronKey/
sha256sum -c ~/CyberHygiene_Recovery_Image_2025-12-18.tar.gz.sha256
```

### Option B: Extract to IronKey
```bash
# Create recovery directory on IronKey
mkdir -p /media/IronKey/recovery-image

# Extract archive to IronKey
tar xzf ~/CyberHygiene_Recovery_Image_2025-12-18.tar.gz -C /media/IronKey/

# Verify extraction
ls -la /media/IronKey/recovery-image/
```

## Step 4: Secure Storage

1. **Safely Eject IronKey:**
   - Sync filesystem: `sync`
   - Unmount: `umount /media/IronKey`
   - Lock IronKey via software interface
   - Physically remove USB drive

2. **Store Securely:**
   - Place IronKey in locked safe or secure facility
   - Document storage location in security log
   - Limit access to authorized administrators only
   - Consider creating duplicate for offsite storage

3. **Security Classification:**
